# web

Interface administrativa via navegador.

## O que ele deve ser?
Um servidor HTTP leve para gerenciar o servidor de forma visual e remota.

## O que precisa fazer?
- [ ] **Dashboard**: Exibir uso de CPU, RAM e disco.
- [ ] **Configuração**: Permitir alterar configurações de rede e serviços pelo browser.
- [ ] **Logs Viewer**: Interface para ler os logs do `logger`.
- [ ] **Segurança**: Autenticação forte e HTTPS obrigatório.
